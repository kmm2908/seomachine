⑤ Quality Gate AUTOMATED · UP TO 2 REWRITES

✅ Checks Run on Every Article

Mandatory (all types)
  • ›Hook quality — no generic "X is a…" openers
  • ›≥ 2 CTAs, first within 500 words (body only)
  • ›≤ 3 long paragraphs (>3 sentences) in body
Optional (need ≥1 of 2)
  • ›Mini-stories — client scenarios with outcome verbs
  • ›Sentence rhythm — varied lengths, low monotony
Readability (Flesch)
  • ›Default / blog / topical: ≥ 55
  • ›Location: ≥ 50 (place names drag score)
  • ›Comp-alt / problem / news: ≥ 48
  • ›News: hook moved to optional

✓ PASS → Proceed to Publish

  • All mandatory criteria met
  • Sufficient optional criteria met
  • Flesch score above threshold for content type
  • Article saved to content/[abbr]/[type]/[slug]/

Publishing
⑥ Publishing WORDPRESS REST API

📤 Pre-publish Steps

  • Schema tokens replaced: [DATE], [BUSINESS_PHONE], logo, price range
  • CSS classes injected: h2→hdr-l, p→txt-m, etc.
  • No-commas/full-stops rule applied to title
  • Images uploaded to WP media library
  • [BANNER_IMAGE_URL] replaced with real WP URL in schema
  • First image set as featured image

🏗️ Elementor Injection

  • Template auto-refreshed if WordPress version is newer
  • HTML injected into "Paste HTML Here" widget
  • First H2 stripped (template has H1 title widget)
  • Schema <script> appended directly
  • Two-section mode (SDY/GTB): S1 body → S1 widget, FAQ → S2 widget
  • Single-section mode (GTM): fallback to first HTML widget
  • Post type set from content_type_map in config

🗂️ Post Created

  • Correct CPT: seo_location, seo_service, etc.
  • Blog / news → standard post type
  • WP category assigned if set in queue
  • Excerpt set = topic/address (powers hub shortcode display)
  • SEO meta fields written (Yoast-compatible keys)
  • Published as draft by default

Site Structure HOW CONTENT FITS THE WORDPRESS SITE

🏠 Hub Pages (Pillar)

  • GBP category landing pages — e.g. "Thai Massage Glasgow"
  • 700–1000 words, broad keyword targeting
  • [seo_hub type="service"] shortcode lists all services
  • [seo_hub type="location"] lists all locations
  • Hub auto-updates as new posts are published

📍 Service + Location Pages

  • Service: each treatment described in 400–600 words
  • Location: neighbourhood/district pages 450w+
  • Both feed into hub shortcodes automatically
  • CPTs: seo_service, seo_location

✍️ Blog + News + Topical

  • Blog / news → standard WP post type
  • Appears in category archives, RSS, theme templates
  • Blog subdomain (GTB) uses cross-site hub to pull CPT content from main site
  • AI brand positioning injected for blog + topical only

⚔️ Comp-Alt + Problem

  • Comp-alt: competitor comparison pages — includes directions widget
  • Problem: condition/symptom pages — mandatory outbound links to NHS/Wikipedia/PubMed
  • Both: 3-column grid on hub via [seo_hub type="problem"]
  • CPTs: seo_comp_alt, seo_problem
